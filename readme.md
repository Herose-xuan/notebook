## 课程收获

通过本教程的学习，你将能够学习到：

- 了解并描述大数据技术的应用场景，真实世界的大数据问题和方法的例子。
- 解释大数据4V的特性以及各种特性如何影响到数据的收集、监控、存储、分析和报告的方方面面。
- 识别大数据问题，并能够将问题转化为数据科学问题。
- 总结HDFS文件系统和MapReduce编程模型的特点和意义，以及它们与大数据的关系。
- 识别和利用数据科学生命周期和相关数据流中的各种组件，如HBase，Hive等。
